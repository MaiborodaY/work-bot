# Рыбалка — Game Design Spec V1

> **Статус:** готово к реализации.
> **Дата:** 2026-04-23.
> **Раздел в игре:** отдельное меню (не привязано к «Дом и Отдых», но тематически близко).

---

## 1. Цель фичи

Подружить игроков, создать повторяющиеся социальные взаимодействия и узнавание друг друга. Механика — итерированная дилемма заключённого (IPD): двое анонимно встречаются на рыбалке, каждый делает выбор **Честно / Жадно**, исходы считаются по матрице. История сессий накапливается и видна в момент каждого выбора.

---

## 2. Основной луп

1. Игрок открывает **Рыбалку** → выбирает водоём → платит ставку → создаёт сессию
2. Анонимный матчинг: второй игрок видит открытую сессию и присоединяется
3. Сессия **стартует немедленно**
4. Игрок 1 получает пуш: «Напарник найден» → видит имя + репутацию + историю → делает выбор **один раз, изменить нельзя**
5. Игрок 2 (уже в приложении) выбирает сразу — тоже один раз
6. Выборы **скрыты друг от друга** до окончания сессии
7. Сессия заканчивается по крону → оба выбора фиксируются → результат приходит обоим одновременно
8. Не сделал выбор до конца сессии → авто **Честно**
9. Партнёр не нашёлся до истечения открытой сессии → ставка возвращается

---

## 3. Водоёмы (тиры)

| Водоём | Ставка | Длительность | Требование |
|---|---|---|---|
| 🏙 Городской пруд | $300 | 3 часа | Уровень рыбака 1 (сразу) |
| 🌲 Загородное озеро | $1 000 | 5 часов | Уровень рыбака 2 (5 сессий) |
| ⛰ Горная река | $5 000 | 8 часов | Уровень рыбака 3 (20 сессий) |

Ставка **всегда возвращается** (Model B). Матрица — только прибыль сверху.

---

## 4. Матрица выплат

### Городской пруд (ставка $300)

| | Партнёр: Честно | Партнёр: Жадно |
|---|---|---|
| **Ты: Честно** | +$600 каждому | +$0 (ставка назад) |
| **Ты: Жадно** | +$1 000 тебе, +$0 ему | +$0 оба |

Итого в кармане: CC = $900 каждому · DC = $1 300 / $300 · DD = $300 оба

### Загородное озеро (ставка $1 000)

| | Партнёр: Честно | Партнёр: Жадно |
|---|---|---|
| **Ты: Честно** | +$2 500 каждому | +$0 |
| **Ты: Жадно** | +$4 000 тебе, +$0 ему | +$0 оба |

Итого: CC = $3 500 · DC = $5 000 / $1 000 · DD = $1 000 оба

### Горная река (ставка $5 000)

| | Партнёр: Честно | Партнёр: Жадно |
|---|---|---|
| **Ты: Честно** | +$12 000 каждому | +$0 |
| **Ты: Жадно** | +$20 000 тебе, +$0 ему | +$0 оба |

Итого: CC = $17 000 · DC = $25 000 / $5 000 · DD = $5 000 оба

### Проверка PD-условий (все тиры ✓)
- DC > CC — жадничать против честного выгоднее одиночно
- CC > DD — взаимная честность лучше взаимной жадности
- 2×CC > DC + 0 — вместе честные зарабатывают больше суммарно

## 5. Уровень рыбака (прогрессия)

Растёт от `completedTotal` — общего числа завершённых сессий **любым исходом** (CC/CD/DC/DD). Хранится в `u.fishing.completedTotal`.

| Уровень | Сессий | Звание | Открывает |
|---|---|---|---|
| 1 | 0 | 🐟 Новичок | Городской пруд |
| 2 | 5 | 🎣 Рыбак | Загородное озеро |
| 3 | 20 | 🏆 Мастер | Горную реку |
| 4 | 50 | 🎖 Ветеран | Статус (новых водоёмов нет) |
| 5 | 100 | 🦈 Легенда | Статус |

Никаких отдельных покупок снаряжения — только уровень.

---

## 6. Репутация

**Что считается:** % честных сессий за последние 10 сессий игрока (все партнёры).

**Что показывается партнёру:**
- Процент: `🟢 80%`
- Визуальная строка последних 10 сессий: `✅✅✅✅❌✅✅✅❌✅`
- До 5 сессий — вместо репутации: `🆕 Новичок`

**Хранение:** `u.fishing.recentOutcomes` — массив последних 10 исходов (C/D моя сторона).

---

## 7. История пары

Хранится в `u.fishing.partnerHistory[partnerId]` — массив объектов `{me: "C"|"D", them: "C"|"D"}` хронологически.

Показывается **всегда** при встрече с партнёром, с которым уже рыбачили:

```
Ваша история:
ты  ✅❌✅
он  ✅✅✅
```

Если рыбачат впервые — блок отсутствует.

## 8. Нотификации

### 8.1 Старт сессии (напарник найден)

```
🎣 Напарник найден!

Рыбачишь с: Степан

📊 Его репутация: 🟢 80%
✅✅✅✅❌✅✅✅❌✅

Ваша история:
ты  ✅❌✅
он  ✅✅✅

[ Честно ]    [ Жадно ]
```

Если первая встреча — блок «Ваша история» отсутствует.
Если партнёр новичок — вместо репутации `🆕 Новичок`.

### 8.2 Конец сессии (результат)

```
🎣 Рыбалка завершена!

Степан: Честно ✅
Ты: Жадно ❌

Улов: +$1 000 💰
Степан остался ни с чем.
```

Тексты результата по исходам:
- **CC:** «Отличная рыбалка! Оба остались в плюсе.»
- **DC (ты жадный):** «Ты забрал больше. Партнёр остался ни с чем.»
- **CD (он жадный):** «Партнёр забрал всё. Ты вернул только ставку.»
- **DD:** «Водоём истощён — оба ушли ни с чем.»

## 9. Ачивки

| ID | Триггер | Название |
|---|---|---|
| `fishing_first_session` | Первая завершённая сессия | Первый заброс |
| `fishing_first_cc` | Первый исход CC | Честные рыбаки |
| `fishing_lake` | completedTotal ≥ 5 | Загородная рыбалка |
| `fishing_river` | completedTotal ≥ 20 | Горная река |
| `fishing_10_sessions` | completedTotal ≥ 10 | Рыбак |
| `fishing_50_sessions` | completedTotal ≥ 50 | Бывалый |
| `fishing_100_sessions` | completedTotal ≥ 100 | Легенда |
| `fishing_partner_5x` | 5+ сессий с одним партнёром | Проверенный напарник |
| `fishing_cc_streak_5` | 5 CC подряд | Честный улов |

События для AchievementService: `fishing_session_completed`, `fishing_cc_result`.

---

## 10. Требования входа и ограничения

- Минимальный уровень игрока: **5** (антисмурф)
- Максимум активных сессий: **1** на игрока (MVP)
- Кулдаун между сессиями: **нет**
- Рыбачить с одноклановцами: **разрешено**
- Очередь на водоём: отдельная для каждого (Пруд / Озеро / Река)

---

## 11. Структура данных (u.fishing)

```js
u.fishing = {
  completedTotal: 0,          // всего завершённых сессий
  recentOutcomes: [],         // последние 10: ["C","D","C",...]  (моя сторона)
  ccStreak: 0,                // текущая серия CC подряд
  activeDeal: "",             // id текущей сессии или ""
  partnerHistory: {           // история по партнёрам
    "<userId>": [
      { me: "C", them: "D" },
      { me: "C", them: "C" }
    ]
  }
}
```

---

## 12. Технические заметки

- Сессии хранятся в KV: `fishing:session:<id>`
- Состояния сессии: `open` → `active` → `finished` | `expired`
- Разрешение через 5-минутный cron (как Синдикат)
- Идемпотентный резолв: проверка `state === "active"` перед записью
- Тайм-аут открытой сессии (партнёр не найден): **24 часа**, возврат ставки
- i18n: все тексты через ключи, не хардкод

---

## 13. Вне скоупа MVP

- Снаряжение с бонусами и расходниками
- Рейтинг / топ рыбаков
- Квесты на рыбалку
- Больше водоёмов
- Сезонные события (зимняя рыбалка и т.п.)
